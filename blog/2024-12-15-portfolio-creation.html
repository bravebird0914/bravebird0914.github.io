ポートフォリオサイトの作成で学んだこと

このポートフォリオサイトを作成する過程で学んだデザインのポイントやアニメーション実装のコツについてまとめました。

はじめに

静岡大学情報学部でNLPの研究をしているbravebirdです。今回、自分のポートフォリオサイトを作成する機会があり、デザインや実装について多くのことを学びました。

デザインのポイント

1. カラースキーム

アクアマリン（#7FFFD4）をメインカラーに選びました。寒色系で清潔感があり、技術系のサイトにマッチします。

2. タイポグラフィ

  • タイトル: Playfair Display（エレガント）
  • 本文: Montserrat（モダン）
  • 日本語: Noto Sans JP

フォントの組み合わせで洗練された印象を作り出しました。

3. アニメーション

.card:hover {
  transform: translateY(-8px);
  box-shadow: 0 16px 32px rgba(127, 255, 212, 0.25);
}

ホバー時のアニメーションで、インタラクティブな体験を提供しています。

実装のコツ

1. CSS変数を活用 → 色の管理が簡単 2. ダークモード対応 → localStorage + prefers-color-scheme 3. レスポンシブデザイン → Grid Layout + Media Queries

まとめ

ポートフォリオサイト作成を通じて、デザインとコーディングの両面で成長できました。今後も改善を続けていきます！
