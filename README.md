## 🎨 カスタマイズ方法

### プロフィール画像の変更
`images/profile.jpg` または `images/profile.png` を配置してください。

### カラースキームの変更
`css/style.css` の以下の変数を編集してください：
- メインカラー: `#7FFFD4` (Aquamarine)
- アクセントカラー: `#00CED1` (Dark Turquoise)
